# The Wacther v2.0
**The Watcher** adalah sebuah program motion-detection surveillance system yang dibangun dengan Python.

## Fitur-Fitur
* Real-time motion detection
* Intruder alarm
* Threat level indicator
* Video Capture

## Requirement 
1. Python Versi 3.6 atau diatasnya
2. opencv
3. imutils
4. numpy
5. pillow
6. ttkbootstrap
